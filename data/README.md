# Dataset: Tamil–English & Hindi–English CodeMix (Knowledge Distillation Project)

This folder contains the complete dataset splits used in the experiments.

Structure:
- data/tamil/
  - train.csv
  - val.csv
  - test.csv
- data/hindi/
  - train.csv
  - val.csv
  - test.csv

Total rows (combined, both languages): ~3600.

Columns:
- text : code-mixed sentence (Tamil-English or Hindi-English)
- label: target/annotation used for training/evaluation
